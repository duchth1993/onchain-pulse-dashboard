## Animation Details

### Entrance Animations
- **Slide Up**: Elements enter from bottom with fade
- **Stagger Delay**: Each item has +50-100ms delay
- **Duration**: 0.5 seconds
- **Easing**: ease-out for natural feel

### Continuous Animations
- **Pulse Glow**: Logo and highlight elements
  - Period: 2 seconds
  - Effect: Expanding blur with color fade
  - Intensity: Subtle to prominent

- **Glow Text**: Primary stat numbers
  - Period: 2 seconds
  - Effect: Text shadow pulsing
  - Range: 0.5x to 0.8x intensity

- **Float**: Decorative elements
  - Period: 3 seconds
  - Effect: Vertical lift and settle
  - Range: Â±10px

### Interactive Animations
- **Hover Cards**: Subtle background glow on hover
- **Number Transitions**: Smooth number counter updates
- **Progress Bars**: Animated fill to target percentage
- **Chart Entries**: Data points appear left-to-right
